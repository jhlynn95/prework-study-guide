HTML

  • Is the backbone of any website
  • The head element contains information about the webpage (not visible to any user)
  • The body element represents the visible content shown to the user
  • There are six different heading tags in HTML. < h1,h2,h3,h4,h5,h6 >
  • It is important to only use one < h1 > as the < h1 > tag will be bigger than the rest
  • Each of the headings decrease in size according to its importance
  • Do not skip any headings by using h1 than jump to h5

CSS

  • CSS (Cascading Style Sheets) is used to style and lay out web pages to alter the font, color, size, and spacing of your content, split it into multiple columns, or add animations and other decorative features.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Border property defines the color, style, and size of the border
  • Content is the heart of the page that is where the information and the images are displayed.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A indicates that we want to add all changes
  • git commit -m "add starter code" describes what the commit contains
  • git pull origin main:
    • identify a git command with git keyword
    • use the pull command to receive a branch's modifications into the local environment
    • origin- keyword indicates the source of the pull will be in the GitHub repo
    • main- keyword indicates the branch
  • git pull origin main- identify a git command with git keyword, use the pull command to receive a branch's modifications into the local environment, origin keyword indicates the source of the pull will be in the GitHub repo, main keyword indicates the branch
  • git push origin "name of branch"- When using the push command, you can create a new remote branch or add new changes to an existing branch.
